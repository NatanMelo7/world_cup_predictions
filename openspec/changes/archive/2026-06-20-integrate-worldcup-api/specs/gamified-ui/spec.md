## ADDED Requirements

### Requirement: XP Celebration Animation
The system SHALL display a celebratory animation when a user earns XP.

#### Scenario: XP awarded after match
- **WHEN** a match concludes and XP is awarded
- **THEN** an animated popup shows the XP amount earned

### Requirement: Level Progress Bar
The system SHALL display a progress bar showing progression toward the next level.

#### Scenario: XP gain updates progress
- **WHEN** user earns XP
- **THEN** the level progress bar animates to reflect new XP percentage

### Requirement: Visual Match Cards
The system SHALL display matches as visual cards with team flags, names, and VS layout.

#### Scenario: Match listing
- **WHEN** user views the predictions page
- **THEN** matches are displayed in a card grid with flags and formatted dates

### Requirement: Gamified Color Palette
The system SHALL use a game-inspired visual theme with dark backgrounds, neon accents, gradients, and rounded cards.

#### Scenario: Game feel
- **WHEN** user navigates any page
- **THEN** the interface uses dark backgrounds, vibrant accent colors, and smooth transitions
